# GeniusGo

Your AI Career Copilot.

GeniusGo helps students and career switchers generate personalized career roadmaps, identify skill gaps, discover projects, and prepare for interviews using AI.

## Features

- Career Roadmap Generator
- Skill Gap Analyzer
- Project Recommender
- Interview Preparation

## Tech Stack

- Next.js
- Tailwind CSS
- OpenRouter AI
- Supabase
- Vercel

## Status

🚧 Currently under development.

## Vision

Help anyone go from confusion to clarity in their career journey with AI-powered guidance.
